Title: Services
---

## Get all services status

```http
GET /tooloop/api/v1.0/services
```


## GET VNC status

```http
GET /tooloop/api/v1.0/services/vnc
```


## Enable VNC

```http
GET /tooloop/api/v1.0/services/vnc/enable
```


## Disable VNC

```http
GET /tooloop/api/v1.0/services/vnc/disable
```
