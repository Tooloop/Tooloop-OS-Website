Title: Keyboard shortcuts
---

| Shortcut           | Description                 |
| :----------------- | :-------------------------- |
| **Helpers**        |                             |
| `Super + t`        | Opens a Terminal            |
| `Super + c`        | Opens a clock               |
| `Super + s`        | Opens the settings          |
| **Windows**        |                             |
| `Super + w`        | Close window                |
| `Super + Esc`      | Send to background          |
| **Screenshots**    |                             |
| `Super + Alt + 3`   | Whole display               |
