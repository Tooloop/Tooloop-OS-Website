# Get installed app

    GET /tooloop/api/v1.0/appcenter/installed


## Get available apps

    GET /tooloop/api/v1.0/appcenter/available


## Re-read app packages

    GET /tooloop/api/v1.0/appcenter/refresh


## Install app

    GET /tooloop/api/v1.0/appcenter/install/<string:name>
